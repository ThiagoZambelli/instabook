Quem Somos

( Seu espaço pessoal para soltar a mente )

Voçê ja teve uma grande ideia textual e não sabia aonde por? Ja teve duvidas sobre um projeto de historia, mas não tinha nenhum leitor Beta? Eu ja! E por esse motivo eu criei o InstaBook.

Este projeto consiste em uma rede social voltada para a troca de textos, contos, poemas entre os mais diversos autores.

Este é um projeto por: Zambelli, Thiago.

Este projeto foi pensado como uma maneira de conciliar um real problema (a falta de lugares confortaves para escritores novatos postarem seus textos) e a necessidade de criação de um portifólio para introdução no mercado de trabalho da programação.
